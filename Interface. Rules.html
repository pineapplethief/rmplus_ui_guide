Общие правила верстки

  1. Отступы контента от контейнера контента (div#content): div#content { padding: 6px 10px 10px 5px; }
  2. Желательно выносить название поля в форме выше самого поля, либо справа от него. Кроме того, если в форме одно поле, то кнопка Сохранить должна располагаться на одном уровне с ним. Желательно, чтобы кнопка и поле были одной высоты. Например:

    Отфильтровать по департаменту

  3. У каждого поля на форме обязательно должен быть сопутствующий label с корректно заполенным аттрибутом for и описанием назначения поля.
  4. При открытии страницы или модального окна с формой фокус должен автоматически переноситься на первый элемент формы. Должна быть возможность переключаться между элементами формы с помощью нажатия tab (см. атрибут tabindex), и отправлять заполненную форму по нажатию клавиши enter.
  5. Если в форме несколько элементов, они должны быть выровнены, например:

  6. Простые элементы формы и аналогичные элементы с Select2 на одной форме должны быть идентичны по ширине, высоте, и т.д.
  7. Запрещено указывать число элементов в счетчике в виде текста в круглых скобках. Вместо этого нужно использовать соответствующий класс счетчика count.
  8. Картинки аватаров пользователей создаются при помощи метода gravatar, размер передается параметром size, возможные значения size: 10, 14, 16, 24, 50, 300 пикселей.
